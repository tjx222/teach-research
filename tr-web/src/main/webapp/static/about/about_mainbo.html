关于明博

明博教育科技股份有限公司创立于2009年，由北大方正集团、新华文轩出版传媒股份有限公司和中国地图出版集团联合发起、共同投资组建。公司依托三大股东在技术研发、内容出版及教育服务等领域的优势资源，以“为教学成就卓越”为使命，致力于为用户提供可持续发展的信息化教学应用生态平台解决方案，旨在成为我国专业权威的教育应用系统软件提供商和服务平台运营商。

公司拥有一支专业的技术研发团队及教研团队，坚持技术创新与产学研深度合作，旗下产品已获得23项软件著作权和软件专利。独立自主研发的“优课”系列产品，为基础教育信息化进程中的学校、教师和学生，提供易用的数字化教学产品和海量的优质教学资源。先后与国内30余家出版社、10余家出版传媒集团及教育集团达成合作，内容上达到版本、学科国内最齐全。

凭借数字化教育内容的优势凸显，2013年明博教育进军在线教育领域，陆续孵化真题馍馍、一起教等互联网教育产品。随着标准化产品和运营类产品的全面上线，明博教育已成为国内整合内容最齐全的教育信息化企业。

明博教育日新月异高速发展，优课用户遍布全国31个省市自治区，150余个地市，服务4.5万余所学校，惠及165万教师用户。在线教育产品“真题馍馍”用户超过100万，被新浪教育中国教育APP测评报告为K12类APP推荐TOP5。明博教育肩负“为教学成就卓越”的使命，致力于打造中国教育信息化软件第一品牌，为中国教育信息化迈向教育现代化而不懈奋斗。
