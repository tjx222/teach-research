三 使用说明

3.4.1 产品介绍

基于“互联网+教育评价”的理念所研制的“评教系统”，充分运用信息技术手段服务学校常态工作，重点解决学校评教操作难、分析统计难的问题。学生、家长等通过网络平台可以及时反馈对学校管理以及教师工作的评价和建议。同时，“评教系统”可以依据大量真实数据统计生成多维度的综合分析报告，为学校改进工作、调整管理策略等提供重要咨询。“评教系统”高度关注学校个性化需求，考核评价指标学校可以自主设定，功能全面，操作简便，是学校做好管理评价的好助手。

其特点为：

（1）流程简约、操作简单、工作量明显降低。

（2）个性化设定评价指标和分值标准。

（3）评价操作人性化，不限制时间、场所。

（4）统计分析功能强大，可生成综合分析报告。
