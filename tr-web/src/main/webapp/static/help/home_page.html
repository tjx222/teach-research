三 使用说明

3.1基础操作

3.1.3 首页

登录后进入首页，显示首页页面（如图3.1-4）

图3.1-4

1、页面头部：

图3.1-5

（1）切换身份：用户点击下拉列表，可以切换至相应身份的工作空间首页。

（2）教研动态：点击“教研动态”按钮可进入通知公告列表页面（如图3.1-6），点击“标题”名称，查看通知公告详情页面。

图3.1-6

（3）消息：点击“我的消息”按钮，可以查看消息列表页面（如图3.1-7），点击“标题”名称，查看消息详情页面，已阅读消息会进入“已读通知”保存。

图3.1-7

（4）退出：点击“退出”按钮，即可安全退出登录。

2、个人信息（如图3.1-8）

显示用户的个人头像、教师姓名和学校名称。点击“个人头像”可以修改个人头像；点击右上角的“设置”按钮，可以“修改个人信息”（ 如图3.1-9）和“修改密码”（如图3.1-10）。

图3.1-8
图3.1-9
图3.1-10

3、课程表及日程表

根据用户身份，页面左下角会显示对应教师的课程表（如图3.1-11）和对应管理者的日程表（如图3.1-13），点击右上角的“设置”按钮，即可修改相关信息（如图3.1-12、图3.1-14）。

4、我的工作项

显示用户常态性的功能模块（如图2.1-15），用户可根据自己的工作需要进行添加工作项和删除工作项。点击各功能模块的名称，直接跳转至相应功能模块页面。
