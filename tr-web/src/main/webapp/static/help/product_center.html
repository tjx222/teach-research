一 产品中心

“学校常态教研与管理平台”从常态化教师教研与学校、区域教学管理出发，为学校、教师提供一站式的应用服务。涵盖教师备课与教学、教师教研、教师培训和学校教学管理绩效考核、统计监管等各方面。

“学校常态教研与管理平台”的特点如下：

（1） 虚拟学校：现实工作项就是功能点，强调工作流程，而不只是简单的栏目堆砌；

（2） 服务常态：只需要做好常态工作，不同的身份做不同的事，发挥信息技术正能量，着眼于提高效率；

（3） 帮忙不添乱：使教师工作更便捷，而不是增加负担，真正解决了“不愿用”的问题；

（4） 少革命多适应：不过多改变教师工作习惯，只需会打字，真正解决了“不好用”的问题；

（5） 自动生成资源：教师的常态工作中自然生成资源，真正解决当前资源平台的资源更新与审核难题。

学校常态教研与管理平台”分为教师工作空间、学校管理空间、区域管理空间。用户可以登录相应的空间，查 看到自己常态的功能模块，并迅速进入功能页面。

“教师工作空间”的功能模块包括：撰写教案、我的备课本、上传课件、教学反思、同伴资源、集体备课、教学 文章、听课记录、计划总结、同伴互助、成长档案袋、区域教研、校级教研。

（1） 撰写教案：

● 调用word文档进行备课，不改变教师使用习惯；

● 随课题同步推送资源，免去查找麻烦，真正实现帮忙不添乱。

（2） 我的备课本：

● 随课题配套收集教案，课件，反思等资源；

● 备课本中，随时交流管理者的查阅意见和他人的听课意见。

（3） 同伴资源：

● 同年级同学科的教师自动集结成群，因备课进度相似，所以可以随教学进度自动生成和更新资源，

确保新鲜而有质量。

（4） 集体备课：

● 教师在这里可以进行“同备教案”、“主题研讨”；

● 教师用碎片时间对集体备课内容留下自己的意见，改变当前学校集体备课效率低、讨论不深入现状。

（5） 同伴互助：

● 将全国同一年级同一学科教师集成群，实现跨区域跨时空的成千上万个“臭皮匠合成诸葛亮”的同伴互 助，找到教改知音。

（6）成长档案袋：

● 自动收集教师网络教研情况，免去教师收集整理档案袋的繁琐和麻烦；不仅有了一个展现自我教学成果的平台，同时也能分享他人精华资源。

“学校管理空间”的功能模块包括：查阅教案、查阅课件、查阅反思、集体备课、查阅计划总结、区域教研、校际教研、管理记录。

（1） 教学检查：随时查阅备课、教研情况，使管理查阅工作日常化。并自动将查阅意见反馈给教师，实现教学管理科研化；

（2） 资源建设：教师教研工作中自动生成资源，并在学校管理过程中确保资源质量，真正资源更新与审核的问题。同时实现将学校已有资源网站的资源，自动推送给教师同步备课，解决用资源的问题。

“区域管理空间”的功能模块包括：区域教研、校际教研。

（1） 用大数据理念设计平台，以学校常态工作自动生成数据为基础，确保数据汇总分析真实准确；

（2） 统一后台管理，模拟教育行业常态工作流程；

（3） 在各学校常态工作基础上开展区域各项教研活动。
